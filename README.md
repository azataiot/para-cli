# para-cli


## What is this about ? 

To manage your [PARA](https://fortelabs.com/blog/para/) methodology in your local file system. 

This includes the following file system tree: 

```
.
├── 00 Inbox
├── 01 Projects
│   └── WQU-SM-M4
├── 02 Areas
├── 03 Resources
│   ├── Alfred
│   └── macOS
└── 04 Archives
```
